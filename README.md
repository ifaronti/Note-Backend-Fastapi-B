### NOTE TAKING FULLSTACK APP

# Story:
  - Users can:
    - Register an account
    - login
    - create notes
    - modify notes
    - archive notes
    - restore notes from archive to active
    - delete notes
    - search notes by tags, content or title
    - view notes by tags
    - change password
    - recieve email for password reset incase forgotten
    - reset password
